Aseprite 简易在线编译教程

Aseprite 是顶级的像素绘画与动画软件。由于其开源特性，我们可以通过 GitHub Actions 在线自动编译，无需在本地安装复杂的开发环境，即可获得与原版一致的体验。

📌 准备工作

  • 一个 GitHub 账号。
  • 网络环境畅通。

第一步：Fork 项目仓库

  1. 登录 GitHub，访问项目：aseprite-builder
  2. 点击页面右上角的 Fork 按钮。
  3. 在跳转页面点击 Create fork，将项目复制到你的个人账号下。

第二步：配置编译参数

  1. 进入你账号下的 aseprite-builder 仓库。
  2. 点击 Actions 选项卡，点击绿色按钮 “I understand my workflows, go ahead and enable them” 启用工作流。
  3. 修改系统匹配（重要）：
  • 依次点击 workflows -> build.yml 文件。
  • 点击右上角 铅笔图标 进入编辑模式。
  • 找到第 55 行，将 os: [windows-latest, ubuntu-latest, macos-latest] 修改为你需要的系统（例如 Windows）：

os: [windows-latest]

  • 点击 Commit changes… 保存修改。

第三步：开始在线编译

  1. 再次点击顶部 Actions 菜单。
  2. 在左侧栏选择 Build and release Aseprite。
  3. 点击右侧的 Run workflow 下拉框，点击绿色的 Run workflow 按钮。
  4. 等待： 编译过程大约需要 15 分钟。当列表出现绿色对勾时，表示编译成功。

第四步：下载与运行

  1. 回到仓库主页（Code 选项卡）。
  2. 在右侧 Releases 栏目中点击最新生成的版本。
  3. 下载对应的 .zip 压缩包并解压。
  4. 运行文件夹内的 aseprite.exe 即可开始使用。

🛠 常见问题处理

报错：找不到 libcrypto-1_1-x64.dll

  • 原因： 系统缺少 OpenSSL 运行库。
  • 解决： 1. 在项目 Release 页面下载 openssl-1.1.1w.zip。
  1. 解压后将 x64/bin/ 目录下的 libcrypto-1_1-x64.dll 复制到 Aseprite 软件根目录（与 .exe 同级）即可。
---

💡 温馨提示

  • 清理： 编译完成后，建议根据用户协议自行删除账户内的 Release 文件。
  • 汉化/主题： 可直接使用 Steam 社区或插件市场的语言包进行导入。
  • 支持正版： 本教程仅供技术交流，有条件的请前往 Steam 购买支持开发者。
