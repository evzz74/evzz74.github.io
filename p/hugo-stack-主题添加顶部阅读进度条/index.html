Hugo Stack 主题添加顶部阅读进度条

这篇文章记录一下如何在 Hugo Stack 主题的文章页中添加一个顶部阅读进度条。实现方式很简单：在文章模板 single.html 中加入 HTML 的 <progress> 标签，再通过 CSS 固定到页面顶部，最后用 JavaScript 根据页面滚动距离动态更新进度。

修改文件

需要修改的文件是：

1

themes/hugo-theme-stack/layouts/_default/single.html

这个文件是 Hugo Stack 主题中文章详情页的模板。也就是说，把代码放在这里之后，进度条会显示在文章页面，而不是首页、归档页或分类页。

添加 progress 标签

找到 single.html 中的 {{ define "main" }}：

1
2

{{ define "main" }}
    {{ partial "article/article.html" . }}

在文章内容渲染前添加：

1

<progress id="content_progress" value="0"></progress>

修改后大致如下：

1
2
3
4

{{ define "main" }}
    <progress id="content_progress" value="0"></progress>

    {{ partial "article/article.html" . }}

这里使用 id="content_progress"，方便后面的 CSS 和 JS 精准选中这个进度条。

添加 CSS 样式

继续在 single.html 中加入下面的样式：

<style>
    #content_progress {
        position: fixed;
        left: 0;
        top: 0;
        z-index: 1000;
        width: 100%;
        height: 4px;
        appearance: none;
        -webkit-appearance: none;
        -moz-appearance: none;
        border: none;
        background-color: transparent;
        color: #0095ea;
    }

    #content_progress::-webkit-progress-bar {
        background-color: transparent;
    }

    #content_progress::-webkit-progress-value {
        background-color: #0095ea;
    }

    #content_progress::-moz-progress-bar {
        background-color: #0095ea;
    }
</style>

几个关键点：

  • position: fixed：让进度条固定在浏览器顶部。
  • top: 0 和 left: 0：让进度条从页面左上角开始。
  • width: 100%：让进度条横跨整个页面宽度。
  • height: 4px：设置进度条粗细。
  • #0095ea：进度条颜色，这里使用蓝色。

如果想让进度条更粗，可以把 height: 4px 改成 6px 或 8px。

<script>
    (function () {
        function initContentProgress() {
            var progressBar = document.querySelector('#content_progress');
            if (!progressBar) {
                return;
            }

            function updateProgress() {
                var maxScroll = document.documentElement.scrollHeight - window.innerHeight;
                progressBar.max = Math.max(maxScroll, 0);
                progressBar.value = window.scrollY;
            }

            updateProgress();
            document.removeEventListener('scroll', updateProgress);
            document.addEventListener('scroll', updateProgress, { passive: true });
            window.removeEventListener('resize', updateProgress);
            window.addEventListener('resize', updateProgress);
        }

        if (document.readyState === 'loading') {
            document.addEventListener('DOMContentLoaded', initContentProgress);
        } else {
            initContentProgress();
        }

        document.addEventListener('pjax:complete', initContentProgress);
    })();
</script>

这段代码做了三件事：

  • 页面加载完成后，初始化阅读进度条。
  • 页面滚动时，根据 window.scrollY 更新进度。
  • 页面高度变化时，重新计算最大滚动距离。

最后一行：

1

document.addEventListener('pjax:complete', initContentProgress);

是为了兼容 Hugo Stack 主题中可能启用的 PJAX 页面切换。如果博客没有启用 PJAX，这一行保留也不会有影响。

效果

保存文件后重新运行 Hugo：

1

hugo server

打开任意文章页，页面顶部就会出现一条蓝色阅读进度条。向下滚动文章时，进度条会从左到右逐渐填满。

如果部署后没有看到效果，可以优先检查这几项：

  • 浏览器是否加载了最新页面，可以强制刷新或清理缓存。
  • 修改的是不是当前正在使用的主题目录。
  • 代码是否放在 {{ define "main" }} 内部。
  • 如果开启了 PJAX，切换文章后是否触发了 pjax:complete。
